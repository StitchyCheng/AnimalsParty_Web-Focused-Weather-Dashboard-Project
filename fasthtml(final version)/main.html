Main entry point for the Weather Dashboard application.
 
This module initializes and runs the FastHTML application, which provides a user interface for
viewing weather data, forecasts, and air quality information. The application includes features such as
user authentication, theme switching, and interactive charts.
 
Attributes:
    API_KEY (str): The API key for accessing weather data services.
    STYLES (Style): A collection of CSS styles for the application's user interface.
 
Functions:
    fast_app(debug, pico, hdrs, secret_key): Initializes the FastHTML application with the given settings.
    login_required(f): A decorator that checks if the user is logged in before allowing access to a route.
    get_login(), post_login(request, username, password): Handles user login functionality.
    get_register(), post_register(username, password, confirm_password): Handles user registration functionality.
    get_dashboard(request): Displays the main dashboard with weather information and forecasts.
    get_search(request, city): Handles searching for weather data by city.
    get_logout(request): Logs out the current user.
    get_root(request): Redirects to the login page if the user is not authenticated.
 
Classes:
    None
 
Routes:
    - /login: Displays the login page and handles login requests.
    - /register: Displays the registration page and handles registration requests.
    - /dashboard: Displays the main dashboard with weather information and forecasts.
    - /logout: Logs out the current user.
    - /: Redirects to the login page if the user is not authenticated.
